Setting up GDScript Formatter in VSCode under Windows 11

2024-02-22

  1. Install python 3.12+.

  2. Install python-setuptools. In Windows Powershell, run: pip install setuptools

  3. Install gdtoolkit in Windows Powershell by running: pip install "gdtoolkit==4.*"

    If using Godot 3.*, run: pip install "gdtoolkit==3.*"

  4. Install vscode.

  5. Go to View > Extensions. Install the following:

    • Python
    • GDScript Formatter by Razoric

  6. Open the settings.json by pressing, Ctrl + Shift + P, and typing Open User Settings.

  7. Click on, Preferences: Open User Settings (JSON).

  8. Add the following entries:

{
	// ...other entries. Make sure there's a comma after these entries.
    "[gdscript]": {
        "editor.defaultFormatter": "Razoric.gdscript-toolkit-formatter"
        },
	"editor.formatOnSave": true
}
  1. The formatter is now setup.
